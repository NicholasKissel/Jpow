# Adding JPow Image

To add a local image of Jerome Powell:

1. **Download or save a photo** of Jerome Powell (official portrait preferred)
2. **Name the file** one of these:
   - `jpow.jpg` (recommended)
   - `jpow.png`
   - `jpow.webp`
3. **Place the file** in the project root directory (same folder as `index.html`)

The website will automatically use the local file if it exists, and fall back to online sources if not found.

## Recommended Image
- Official portrait from Wikipedia Commons or similar
- Portrait orientation (vertical)
- Good quality/resolution
- File size: under 2MB recommended

## Where to get the image
- Wikipedia: https://commons.wikimedia.org/wiki/File:Jerome_Powell_official_portrait.jpg
- Or search for "Jerome Powell official portrait" and save the image
